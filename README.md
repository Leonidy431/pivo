# 🌿 Терапия - Здоровые травяные рецепты

React приложение для поиска и управления терапевтическими травяными рецептами, специально разработанное для людей с подагрой и диабетом.

## ✨ Особенности

- **📖 99 Терапевтических рецептов** - безопасные при подагре и диабете
- **📍 Поиск магазинов по GPS** - найдите ближайшие магазины с нужными ингредиентами
- **❤️ Система избранного** - сохраняйте любимые рецепты
- **👤 Профиль пользователя** - управляйте предпочтениями и аллергиями
- **🔍 Поиск и фильтры** - быстрый поиск по категориям и показаниям
- **📱 Мобильное-первое** - оптимизировано для всех устройств

## 🚀 Быстрый старт

### Требования
- Node.js 16+ 
- npm или yarn

### Установка

```bash
# Установите зависимости
npm install

# Скопируйте .env файл
cp .env.example .env

# Обновите API URL в .env если необходимо
# VITE_API_BASE=http://localhost:8000/api/v1
```

### Разработка

```bash
# Запустите dev сервер
npm run dev

# Приложение будет доступно на http://localhost:5173
```

### Продакшен

```bash
# Соберите приложение
npm run build

# Проверьте сборку локально
npm run preview
```

## 🔌 API Интеграция

Приложение подключается к backend API (K-Lab Platform):

```
GET  /api/v1/herbal/recipes              # Список рецептов
GET  /api/v1/herbal/recipes/{id}         # Детали рецепта
GET  /api/v1/herbal/recipes/search       # Поиск рецептов
GET  /api/v1/herbal/shops                # Список магазинов
GET  /api/v1/herbal/shops/{id}           # Детали магазина
GET  /api/v1/herbal/shops/near           # Магазины рядом
GET  /api/v1/herbal/shops/city/{city}    # Магазины в городе
GET  /api/v1/herbal/recipes/{id}/available  # Где найти рецепт
```
